# 15min — обзор для агента

Детерминированный генератор русских математических задач без LLM:
15-минутки (production-каталог), архивный корпус с recovery-ответами и
отдельный олимпиадный трек (239). Сначала собирается структурированная модель
задачи, потом рендер в русский текст.

**Истина — git.** При расхождении БЗ и репозитория верен репозиторий.
БЗ — накопительная память между агентами: маршруты, разборы регрессий, журнал.

## Минимальный стартовый набор чтения (экономит токены)

1. `AGENTS.md` — обязательные правила (ветка от main, запреты, DoD).
2. Хвост `Docs/WORK_LOG.md` — что менялось последним.
3. `Docs/AGENT_STATUS.md` — какая тематическая группа занята.
4. Если работа с шаблонами — `Docs/TEMPLATE_AUTHORING_GUIDE.md` (выбор одного
   из трёх маршрутов — обязателен).
5. Точечный маршрут до файлов — `Docs/VECTOR_TREE.md` (не искать вслепую).

## Команды

- тесты: `python -m unittest discover -s tests` (Windows: `py -3 -m unittest ...`);
- сайт: `python scripts/run_site.py`;
- одна задача: `python scripts/generate_problem_from_template.py --module <m> --difficulty <d>`.
