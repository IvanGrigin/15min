# Карта проекта

## Слои (канон из AGENTS.md)

- `Docs/` — документация + read-only корпус `Docs/all_tasks_all_files.md` (не менять);
- `data/` — схемы, словари, каталоги шаблонов (`data/templates/`, `data/source_index/`);
- `problemgen/` — вся переиспользуемая логика (генерация, язык, веб, worksheet);
- `scripts/` — только entry point-ы; `outputs/` — только артефакты; `tests/` — проверки.

## Три изолированных маршрута авторинга (не смешивать)

| Маршрут | Каталог | Логика |
|---|---|---|
| Production 15-минутки | `data/templates/problem_templates.json` | `problemgen/generation/template_generator.py` |
| Архивный recovery (overlay, архив неизменен) | `data/templates/all_tasks_answer_recovery.json` | `problemgen/worksheet/all_tasks_site.py` |
| Олимпиадный трек 239 | `data/templates/olympiad_templates.json` | модули `olymp_*` |

Плюс тематические наборы: `data/templates/problem_sets/<id>/templates.json`
(+ генератор в `problemgen/generation/<id>_templates.py`) — см. «20 Семейства шаблонов».

## Ключевые инварианты

- шаблон неизменяем: генератор только подставляет значения в `template_text`;
- `number_strategy` обязана возвращать значения внутри `constraints`
  (рассинхрон = хрупкость, см. «30 Регрессии / 205 хрупких шаблонов»);
- `answer_formula` — безопасный вычислитель с белым списком функций
  (`_FUNCTIONS` в `template_generator.py` — штатная точка роста);
- у каждой русской словоформы ровно 3 формы (ед/дв/мн).
