# problemgen/catalog

Здесь должен жить слой каталога шаблонов.

## Задачи слоя

- читать `data/source_index/`;
- читать `data/templates/`;
- строить индекс шаблонов;
- искать шаблоны по фильтрам.

## Примеры будущих функций

- `load_source_index()`
- `load_template_catalog()`
- `find_templates_by_filters()`
- `resolve_template_by_id()`
