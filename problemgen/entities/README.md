# problemgen/entities

Здесь должен жить слой работы с сущностями.

## Задачи слоя

- загружать словари из `data/entities/`;
- выбирать сущности по роли и ограничениям;
- проверять совместимость сущности и шаблона;
- отдавать структурированные entity binding, а не готовые строки.

## Примеры будущих функций

- `load_entity_repository()`
- `pick_entity_for_role()`
- `validate_entity_binding()`
