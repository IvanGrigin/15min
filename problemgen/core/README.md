# problemgen/core

Здесь должны лежать самые общие модели и типы проекта.

Примеры того, что здесь допустимо:

- `TemplateSpec`;
- `TaskInstance`;
- `GenerationBundle`;
- `FeatureFlags`;
- `SolutionProfile`;
- общие перечисления сложности и режимов.

Здесь не должно быть:

- web-кода;
- больших словарей сущностей;
- бизнес-логики отдельных доменов.
