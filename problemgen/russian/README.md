# problemgen/russian

Это переходный совместимый слой русского языка.

## Что здесь допустимо

- существующие простые утилиты согласования;
- временные адаптеры старой архитектуры;
- минимальные проверки уже работающих доменов.

## Что нужно делать дальше

Новые крупные механизмы морфологии, синтаксиса и рендеринга проектировать в `problemgen/language/`, а не расширять бесконечно этот переходный слой.
