# problemgen/domains

Здесь лежат доменные адаптеры.

## Что должен делать домен

- объявлять свое имя и метаданные;
- знать, какие семейства шаблонов к нему относятся;
- уметь фильтровать их по параметрам;
- запускать генерацию через общие сервисы.

## Чего домен не должен делать

- хранить в себе общий словарь русского языка;
- дублировать общий рендеринг;
- быть единственным местом, где известна структура итогового JSON.
