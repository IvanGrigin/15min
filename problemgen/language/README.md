# problemgen/language

Это целевой основной языковой слой проекта.

## Подслои

- `morphology/` — словоформы и грамматические признаки;
- `syntax/` — управление и валентность;
- `renderer/` — сборка финальных строк;
- `validators/` — проверка правильности текста и trace-данных.

## Главная цель

Сделать так, чтобы задачи формулировались не вручную в строках, а через формальные правила и рендеринг.
