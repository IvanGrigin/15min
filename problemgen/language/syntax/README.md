# problemgen/language/syntax

Здесь должен жить слой синтаксических рамок.

## Задачи слоя

- хранить управление глаголов;
- выбирать нужные предлоги;
- определять, какой падеж требуется слоту по роли.

## Примеры будущих функций

- `resolve_verb_frame()`
- `resolve_required_case_for_role()`
- `resolve_preposition_for_location()`
