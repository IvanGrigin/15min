# problemgen/language/morphology

Здесь должен жить слой морфологии.

## Задачи слоя

- выбирать падежные формы сущностей;
- учитывать род, число и одушевленность;
- работать со словарями из `data/entities/` и `data/language/`.

## Примеры будущих функций

- `resolve_case_form()`
- `resolve_number_form()`
- `resolve_gender_dependent_form()`
