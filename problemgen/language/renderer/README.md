# problemgen/language/renderer

Здесь должен жить рендеринг финального текста.

## Задачи слоя

- собирать текст условия из структурированных сегментов;
- собирать текст ответа;
- вести trace, какие формы и правила были использованы.

## Примеры будущих функций

- `render_problem_text()`
- `render_answer_text()`
- `build_language_trace()`
