# problemgen/worksheet

Этот слой отвечает за сборку ученических листов из уже существующих генераторов задач.

## Что здесь должно жить

- валидация уровней сложности для 5 задач;
- выбор задач через существующие доменные генераторы;
- подготовка JSON с условиями и отдельного JSON с ответами;
- вызов рендера листа по JSON-шаблону;
- возврат путей к итоговым артефактам.

## Что здесь не должно жить

- низкоуровневая отрисовка PDF/PNG;
- HTML и JavaScript сайта;
- хардкод готовых задач.

## Главный модуль

- `service.py` — orchestration-слой между генератором задач, JSON-шаблоном и готовым ученическим листом.
