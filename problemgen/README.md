# problemgen

Это основная папка Python-кода генератора.

## Общий принцип

Код здесь должен быть разделен на слои:

- `core/` — общие модели и базовые типы;
- `catalog/` — загрузка и фильтрация шаблонов;
- `entities/` — словари сущностей и выбор по ролям;
- `language/` — новый языковой слой;
- `generation/` — сборка конкретного экземпляра задачи;
- `domains/` — доменные адаптеры;
- `io/` — чтение и запись данных;
- `web/` — локальный HTTP-слой.

## Важное правило

Если новый код начинает работать с данными, уже описанными в `data/`, он должен читать их через явный слой загрузки, а не через дублирование констант.
