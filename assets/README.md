# assets

В этой папке лежат изображения, которые автоматически попадают в листы с задачами.

Сейчас используются:

- `logo.png` — логотип для правой панели листа;
- `logo_239.png` — фирменный знак «Поступление в 239» для интерфейса и
  актуального печатного шаблона;
- `qr.png` — QR-код для правой панели листа.
- `fonts/Lato-Regular.ttf`, `fonts/Lato-Bold.ttf`, `fonts/Lato-Black.ttf` —
  локальная кириллическая версия Lato для интерфейса и печати;
- `fonts/OFL.txt` — лицензия SIL Open Font License 1.1 для Lato.

Шрифты отдаются локальным сайтом из `problemgen/web/worksheet_site.py`, поэтому
лист сохраняет типографику и при печати без внешнего доступа к интернету.

## Как заменить изображения

1. Сохранить новый файл поверх `logo.png` или `qr.png`.
2. Либо изменить пути в `data/templates/worksheets/worksheet_5_tasks.json` в блоке `assets.defaults`.

Важно:

- изображения не должны хардкодиться в Python-коде;
- путь к ним задается через JSON-шаблон листа.
