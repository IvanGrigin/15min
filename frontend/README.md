# frontend

Здесь лежит браузерная часть локального интерфейса.

## Что должно быть в этой папке

- стили;
- клиентский JS;
- при необходимости маленькие вспомогательные ресурсы для UI.

## Что не должно быть в этой папке

- доменная логика;
- словари сущностей;
- правила русского языка;
- генерация числовых параметров.
