# frontend

Здесь лежит браузерная часть локального интерфейса.

## Что должно быть в этой папке

- стили;
- клиентский JS;
- при необходимости маленькие вспомогательные ресурсы для UI.

`worksheet_site.js` нумерует все доступные модули в ручном selector в порядке,
который возвращает `/api/templates`; номер помогает соотнести пункт выбора с
модульным каталогом и не меняет его `module_id`.

## Что не должно быть в этой папке

- доменная логика;
- словари сущностей;
- правила русского языка;
- генерация числовых параметров.
