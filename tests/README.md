# Тесты

Здесь должны лежать проверки генераторов.

Целевая структура:

- `tests/unit/` — маленькие локальные проверки функций и сервисов;
- `tests/integration/` — полные сценарии генерации;
- `tests/golden/` — эталонные тексты и JSON;
- `tests/fixtures/` — тестовые входные данные.

Для новых задач желательно добавлять хотя бы такие тесты:

- ответ всегда целый;
- текст условия не пустой;
- диапазоны чисел не выходят за ожидаемые границы;
- генератор создает нужное количество задач.

Для нового языкового слоя дополнительно желательны проверки:

- падежная форма существует и подставляется корректно;
- глагольная форма согласована с субъектом;
- в финальном тексте нет необработанных плейсхолдеров;
- feature flags и JSON-метаданные не противоречат содержимому задачи.
