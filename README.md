# 15min

Генератор математических задач с модульной архитектурой, отдельными доменами и служебной документацией для дальнейшего роста проекта.

Ключевая структура:

- `problemgen/` — переиспользуемая логика генерации;
- `scripts/` — запускаемые вручную скрипты;
- `outputs/` — сгенерированные артефакты;
- `docs/` — навигация и журнал проекта;
- `tests/` — будущие проверки генераторов.
