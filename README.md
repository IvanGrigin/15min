# 15min

Генератор математических задач с модульной архитектурой, отдельными доменами, расширяемым слоем данных и подробной агентской документацией.

## Что важно сейчас

Проект находится в переходном состоянии:

- текущий рабочий код лежит в `problemgen/`;
- исходный read-only корпус задач лежит в `Docs/all_tasks_all_files.md`;
- целевая архитектура описана в `Docs/ARCHITECTURE_BLUEPRINT.md`;
- путь данных описан в `Docs/DATA_FLOW.md`;
- каркас новых слоев уже размечен через `data/` и новые подпакеты в `problemgen/`.

## Главные папки

- `Docs/` — архитектура, журнал, навигация и read-only корпус задач.
- `data/` — схемы, шаблоны, сущности, таксономии и языковые словари.
- `problemgen/` — код генератора, каталога, сущностей, языка и сборки задач.
- `scripts/` — ручные entry point-файлы.
- `outputs/` — сгенерированные артефакты.
- `tests/` — unit, integration, golden и fixtures.

## Дополнительно по текущей реализации

- `problemgen/core/story_worlds.py` — единый слой сюжетных миров, локаций и персонажей для новых генераторов.
- `problemgen/domains/olympiad_logic/` — домен олимпиадных логических задач с отдельными шаблонами, солверами и валидаторами.
