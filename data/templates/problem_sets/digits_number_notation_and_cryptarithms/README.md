# Цифры, запись чисел и криптарифмы

Единый набор шаблонов по read-only источникам `docs/08a_...md` и
`docs/08b_...md`. Все номера источников учитываются ровно один раз в
`templates.json`; неподдерживаемые сайтом доказательные и игровые форматы
сохраняются как неактивные записи с причиной исключения.

Генератор: `problemgen/generation/digits_templates.py`.
Проверка: `py -3 scripts/validate_digits_templates.py`.
