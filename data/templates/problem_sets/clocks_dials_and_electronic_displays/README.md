# Часы, циферблаты и электронные табло

Набор `clocks_dials_and_electronic_displays` содержит parameterized-задачи
модуля 16. `templates.json` хранит активные integer-шаблоны, а
`source_accounting.json` фиксирует все 25 deduplicated-источников.

Генератор: `problemgen/generation/clock_templates.py`.
Проверка: `py -3 scripts/validate_clock_templates.py`.
