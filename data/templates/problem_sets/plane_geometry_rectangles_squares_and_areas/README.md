# Модуль 24: планиметрия прямоугольников, квадратов и площадей

`templates.json` содержит 18 active exact-integer семейств, а
`source_accounting.json` — полный учёт 75 уникальных номеров двух read-only
источников: 44 active и 31 исключение с точной причиной.

Каталог читает `problemgen/generation/plane_geometry_templates.py`; модуль
зарегистрирован в `problemgen/web/worksheet_site.py` и выбирается сайтом как
одна из пяти позиций листа.
