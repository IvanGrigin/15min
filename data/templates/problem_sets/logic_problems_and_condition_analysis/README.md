# Логические задачи и анализ условий

Набор `logic_problems_and_condition_analysis` использует два read-only источника
`docs/30_logicheskie_zadachi_i_analiz_usloviy_*_deduplicated.md`. В них 37
уникальных задач: 9 активных исходных номеров сведены к трём exact-integer
семействам, остальные учтены в `source_accounting.json` с причиной исключения.

`templates.json` содержит только активные шаблоны. Генератор строит связанные
параметры для одной ошибочной таблицы умножения, исчерпывающе проверяет круг
рыцарей и лжецов и независимо доказывает единственность правдивца. Именованное
семейство выбирает трёх разных утверждённых персонажей одной вселенной и не
склоняет имена эвристически.

Проверки: `py -3 -m unittest tests.test_logic_templates` и
`py -3 scripts/validate_logic_templates.py`.
