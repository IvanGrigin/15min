# data/templates/worksheets

Здесь лежат JSON-шаблоны оформления листов с задачами.

## Что хранится в шаблоне

- размер страницы;
- отступы;
- линии и блоки заголовка;
- слоты для задач;
- правые панели под изображение, логотип или QR-код;
- плейсхолдеры для подстановки данных.

## Что здесь не хранится

- сами сгенерированные задачи;
- Python-логика рендера;
- бизнес-логика генерации математики.

## Кто использует этот слой

- `problemgen/io/worksheet_renderer.py` загружает шаблон и строит лист;
- `scripts/render_worksheet.py` служит ручной точкой запуска.
