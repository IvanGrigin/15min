# data/templates/arithmetic

Здесь должны лежать шаблоны арифметических задач.

Примеры будущих семейств:

- простое сложение и вычитание;
- неизвестный компонент;
- текстовые задачи на части и целое;
- задачи на стоимость, длину, количество.

Примеры будущих loader-функций:

- `load_arithmetic_templates()`
- `find_arithmetic_templates_by_difficulty()`
