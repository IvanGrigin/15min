# Словарь существительных / russian_nouns.json

Файл `russian_nouns.json` — главная библиотека существительных для шаблонов задач.

## Как добавить слово

Добавьте новую запись в `"nouns"` в `russian_nouns.json`. Python-код менять не нужно — словарь загружается автоматически.

Пример записи:

```json
"кролик": {
  "gender": "m",
  "animate": true,
  "legs": 4,
  "tags": ["animal"],
  "forms": {
    "nom": "кролик", "gen": "кролика", "dat": "кролику",
    "acc": "кролика", "ins": "кроликом", "pre": "кролике",
    "nom_pl": "кролики", "gen_pl": "кроликов", "dat_pl": "кроликам",
    "acc_pl": "кроликов", "ins_pl": "кроликами", "pre_pl": "кроликах"
  }
}
```

## Поля

| Поле | Тип | Описание |
|------|-----|----------|
| `gender` | `"m" / "f" / "n"` | Грамматический род |
| `animate` | bool | Одушевлённость |
| `legs` | int / null | Число ног (для задач «головы и ноги») |
| `tags` | list | Категории: `animal`, `food`, `person`, `tool`, `money`, `unit_*`, `game`, `transport`, `place`, `abstract` |
| `count_forms` | `[one, few, many]` | Переопределение форм при счёте, если стандарт (nom / gen / gen_pl) неверен |
| `forms` | object | 12 явных падежных форм (6 ед. + 6 мн.ч.) |

## Падежные ключи

`nom gen dat acc ins pre` + `nom_pl gen_pl dat_pl acc_pl ins_pl pre_pl`

## count_forms

Указывайте только когда стандарт `(nom, gen, gen_pl)` ошибочен:
- `"год"` → `["год", "года", "лет"]` (gen_pl нестандартный: «лет», а не «годов»)
- `"мороженое"` → `["мороженое", "мороженых", "мороженых"]` (форма для 2–4 отличается от gen)
- `"человек"` → `["человек", "человека", "человек"]` (gen_pl: «человек», не «людей»)
