# Шаблоны систем уравнений

## Где лежит модуль

JSON-шаблоны модуля:

`data/templates/problem_sets/systems_of_equations/templates.json`

Источник:

`docs/03_sistemy_uravneniy_updated.md`

Генератор:

`problemgen/generation/system_equation_templates.py`

## Связь с другими модулями

Модуль расположен рядом с наборами `arithmetic` и `equations` в папке:

`data/templates/problem_sets/`

На сайте он подключен как обычная категория задач. Пользователь выбирает модуль `Системы уравнений`, а программа случайно выбирает один шаблон из внутреннего пула.

## Как устроен шаблон

Каждый шаблон описывает структуру коэффициентов:

- фиксированный коэффициент, например `X` или `-X`;
- случайный ненулевой коэффициент;
- коэффициент, равный другому коэффициенту;
- коэффициент, противоположный другому коэффициенту.

Так сохраняются важные свойства исходных задач: одинаковые коэффициенты, противоположные коэффициенты и неявные `1` или `-1`.

## Как гарантируются целые ответы

Генератор сначала выбирает целые значения:

```text
X = x_solution
Y = y_solution
```

Затем выбирает коэффициенты `a`, `b`, `c`, `d` и вычисляет правые части:

```text
e = a × X + b × Y
f = c × X + d × Y
```

После этого система проверяется через определитель:

```text
determinant = a × d - b × c
```

Если определитель равен нулю, вариант отбрасывается.

## Независимая проверка

После построения система решается точными дробями:

```text
X = (e × d - b × f) / determinant
Y = (a × f - e × c) / determinant
```

Генератор принимает вариант только если оба ответа целые и совпадают с выбранными заранее `x_solution` и `y_solution`.

## Отображение знаков

Форматтер не печатает `1X`, `-1X`, `1Y`, `-1Y`.

Примеры:

```text
X + 5Y
X - 3Y
-X + 2Y
-7X - 2Y
```

Некорректные варианты вроде `X + -3Y` запрещены.

## Дубликаты

Задачи `700` и `886` являются точными дубликатами. Они записаны в один шаблон:

`system_equation_006`

Так дубликат не получает двойную вероятность при случайном выборе шаблона.

## Как добавить новый шаблон

1. Добавить исходную задачу в источник или выбрать существующую структуру.
2. Создать запись в `templates.json`.
3. Описать `coefficient_structure`.
4. Указать `source_problem_numbers`.
5. Запустить проверки.

## Команды

Проверить 200 вариантов на каждый шаблон:

```powershell
py -3 .\scripts\validate_system_equation_templates.py
```

Запустить тесты:

```powershell
py -3 -m unittest tests.test_system_equation_templates
```

Запустить сайт:

```powershell
py -3 .\scripts\run_site.py
```
