# Модуль 29A: алфавитный порядок

`alphabetic_order` использует только `29a_alphabetic_order_deduplicated.md` и
`module_29a_regular_human_names.md`. Все 13 номеров распределены между
rank/next/previous/first и repetition-last семействами.

Имя автоматически фильтруется по неповторяющимся буквам и длине 4–5: остаются
13 имён. Шестибуквенные и более длинные имена не выбираются.
Алфавит равен uppercase-буквам имени, включая `Ё` и `Ь`; родительный падеж
хранится в проверенной таблице. Для перестановок число слов равно `n!`, для
повторов — `n^n`. В каждой формулировке явно сказано «Настоящий алфавитный
порядок букв неизвестен» и «выписали по алфавиту». Подсказка сообщает о 5–10-м слове, а не о первом; для
перестановок она полностью фиксирует скрытый порядок. rank/unrank не
материализуют полный список слов.
