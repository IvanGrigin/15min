# Подсчёт целых чисел в промежутках

Каталог находится в `data/templates/problem_sets/integer_interval_counting/templates.json`. Он содержит пять уникальных структур из `docs/06_podschet_tselyh_chisel_v_promezhutkah.md`; одинаковые исходные задания сгруппированы и не повышают вероятность выбора.

Для включённого промежутка число чётных значений вычисляется как `R // 2 - (L - 1) // 2`, число нечётных — по дополнению. Задача «между числами» имеет отдельный тип `exclusive`: обе границы исключаются. Условие на цифру проверяется точным перебором ограниченного промежутка: каждое подходящее число считается один раз, а не по числу вхождений цифры.

Генератор принимает fixed seed, возвращает неотрицательный целый ответ и доступен в общем сайте как «Подсчёт целых чисел в промежутках».

```powershell
py -3 .\scripts\validate_integer_interval_templates.py
py -3 -m unittest tests.test_integer_interval_templates
```

Для нового шаблона добавьте отдельную JSON-запись с непересекающимися `source_problem_numbers`, явно укажите `interval_type` и добавьте стратегию и независимую проверку в генератор.
