# Docs

Папка `Docs/` хранит служебную документацию для будущих агентов и разработчиков.

## Содержимое

- `FILE_INDEX.md` — индекс файлов и их назначение
- `VECTOR_TREE.md` — короткое смысловое дерево навигации
- `WORK_LOG.md` — журнал изменений и выполненных задач

## Как этим пользоваться

Если нужно быстро войти в проект:

1. открыть `AGENTS.md`
2. открыть `Docs/VECTOR_TREE.md`
3. открыть `Docs/FILE_INDEX.md`
4. посмотреть последние записи в `Docs/WORK_LOG.md`

## Зачем это нужно

Эти файлы уменьшают повторный поиск по проекту и помогают следующему агенту быстро понять:

- где лежит нужная логика;
- что недавно менялось;
- какие файлы связаны между собой.
