# Реестр активности авторских веток

Этот файл предотвращает параллельное редактирование одной тематической группы.
Перед началом работы автор шаблонов помечает свою ветку как `в работе`; после
завершения всех её листьев и открытия PR меняет статус на `свободна для
перепроверки`. Статус `свободна для перепроверки` означает: новые шаблоны в этой
группе без отдельного согласования не добавляются; разрешены только ревью и
исправления, найденные при проверке.

| Ветка | Группа | Статус | Владелец | Начато | Прогресс | Примечание |
|---|---|---|---|---|---|---|
| `authoring/D` | D — теория чисел | **в работе** | Codex | 2026-07-15 | 9/9 листьев, 130/130 задач | Авторинг D завершён; ожидается открытие PR. До этого менять листья и шаблоны D нельзя. |
