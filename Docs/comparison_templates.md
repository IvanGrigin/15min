# Шаблоны сравнения чисел и выражений

## Где лежит модуль

JSON-шаблоны:

`data/templates/problem_sets/comparison_of_numbers_and_expressions/templates.json`

Генератор:

`problemgen/generation/comparison_templates.py`

Источники математических структур:

- `docs/04a_sravnenie_bez_imen_i_personazhey.md`
- `docs/04b_sravnenie_s_imenami_i_personazhami.md`

Источник персонажей:

`docs/approved_dimensions_150_characters.md`

## Как объединены источники

Задачи без персонажей и задача с персонажами входят в один общий модуль:

`comparison_of_numbers_and_expressions`

На сайте он отображается как:

`Сравнение чисел и выражений`

Пользователь выбирает модуль, а генератор случайно выбирает один шаблон из общего пула.

## Схема шаблона

Каждый шаблон хранит:

- `source_problem_numbers` — номера исходных задач;
- `source_examples` — исходные примеры;
- `generation_strategy` — имя стратегии в Python-коде;
- `uses_characters` — нужны ли персонажи;
- `required_character_count` — сколько персонажей нужно;
- `tags` и `difficulty` — служебные поля для навигации.

В JSON не хранится исполняемый код. Все производные значения считаются безопасными функциями генератора.

## Математические зависимости

Генератор не выбирает все видимые числа независимо.

Например, для структуры:

```text
a × b
(a + d) × (b - d)
```

сначала выбираются `a`, `b`, `d`, затем производные множители вычисляются из них.

Так сохраняется алгебраический способ сравнения, а не превращение задачи в длинное умножение.

## Точное деление

Для задач с делением сначала выбираются целые частные и делители:

```text
dividend = quotient × divisor
```

Поэтому каждое деление точное, а ответ остается целым.

## Задача про изменение разности

Шаблон `comparison_010` не вводит лишние начальные значения. Ответ считается так:

```text
разность уменьшится на decrease_amount + increase_amount
```

## Персонажи

Персонажный шаблон выбирает:

1. одну утвержденную вселенную;
2. двух разных персонажей из этой вселенной;
3. глаголы прошедшего времени по полу персонажа.

Пример:

```text
Нюша записала ...
Крош записал ...
```

Персонажи из разных вселенных в одной задаче не смешиваются.

## Суммы двузначных чисел

Для задачи с цифрами генератор перечисляет все числа от `10` до `99` и включает число, если нужная цифра встречается хотя бы один раз.

Число `77` учитывается один раз, а не дважды.

## Как добавить шаблон

1. Найти структуру в одном из исходных файлов.
2. Добавить запись в `templates.json`.
3. Добавить стратегию генерации в `comparison_templates.py`.
4. Убедиться, что все производные значения считаются из базовых параметров.
5. Запустить проверки.

## Команды

Проверить 200 вариантов на каждый шаблон:

```powershell
py -3 .\scripts\validate_comparison_templates.py
```

Запустить тесты:

```powershell
py -3 -m unittest tests.test_comparison_templates
```

Запустить сайт:

```powershell
py -3 .\scripts\run_site.py
```
