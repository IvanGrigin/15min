# Шаблоны уравнений

## Что добавлено

В проект добавлен модуль `equations` для генерации задач по теме уравнений и неравенств.

Источник задач:

`docs/02_uravneniya_i_neravenstva_without_1502.md`

Каталог шаблонов:

`data/templates/problem_sets/equations/templates.json`

## Что входит в модуль

- одиночные уравнения с целым ответом;
- неравенства, где ответом является наибольшее натуральное решение;
- системы линейных уравнений с целыми парами ответов;
- задания из нескольких пунктов.

## Как работает генерация

Генератор не берет готовый ответ из исходного файла. Он выбирает целое значение `x`, подставляет его в выражение с `x`, вычисляет правую или левую числовую часть и получает новое корректное уравнение.

Для систем генератор сначала выбирает целую пару `(x, y)`, затем строит правые части уравнений. Поэтому ответы остаются целыми.

## Команды

Пересобрать JSON из исходного файла:

```powershell
py -3 .\scripts\build_equation_templates_from_source.py
```

Проверить шаблоны:

```powershell
py -3 .\scripts\validate_equation_templates.py
py -3 .\scripts\validate_equation_templates.py --seeds 100
py -3 -m unittest tests.test_equation_templates
```

Запустить сайт:

```powershell
py -3 .\scripts\run_site.py
```

На сайте теперь можно выбирать модули `Арифметические вычисления` и `Equations` в любых пяти слотах.
