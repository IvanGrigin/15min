# Делимость, кратные, остатки и простые числа

Каталог `data/templates/problem_sets/divisibility_multiples_remainders_primes/templates.json` объединяет оба исходника. Активны 11 структур; задача 638 учтена, но исключена из выбора как неоднозначная.

Генератор использует целую арифметику: открытые и включённые промежутки различаются формулами, составной делитель строится только как НОК двух поддерживаемых признаков. Для задач с цифрой числа перебираются один раз. Задачи с мёдом строятся из целых числа банок и ёмкости; все персонажи одного условия выбираются из одной вселенной.

```powershell
py -3 .\scripts\validate_divisibility_templates.py
py -3 -m unittest tests.test_divisibility_templates
```
