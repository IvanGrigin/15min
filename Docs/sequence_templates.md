# Последовательности, прогрессии и суммы

Набор находится в `data/templates/problem_sets/sequences_progressions_and_sums/templates.json` и объединяет два источника: `05a` (без персонажей) и `05b` (с персонажами). Все номера источника учитываются ровно один раз; дубликаты объединены в одну запись и не меняют вероятность её выбора.

Генератор `problemgen/generation/sequence_templates.py` использует только целочисленную арифметику. Он поддерживает суммы арифметических прогрессий, чередующиеся шаги и знаки, рекурсии по модулю 10 с обнаружением цикла, циклические сдвиги цифр, продолжение последовательности и составную задачу из трёх частей.

Для задач о подряд идущих числах сначала выбирается интервал, затем вычисляется число цифр. Независимый решатель проверяет единственность первого числа. Персонаж выбирается из одной вселенной файла `docs/approved_dimensions_150_characters.md`; форма «заметил/заметила» определяется по его грамматическому роду.

На сайте модуль называется «Последовательности, прогрессии и суммы» и при выборе случайно выбирает один уникальный шаблон общего пула.

Проверка:

```powershell
py -3 .\scripts\validate_sequence_templates.py
py -3 -m unittest tests.test_sequence_templates
```

Чтобы добавить шаблон, внесите одну JSON-запись с уникальным `id`, стратегией из генератора и непересекающимся списком `source_problem_numbers`, затем дополните стратегию и проверки.
