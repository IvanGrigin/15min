# Арифметические шаблоны

Этот слой формализует задачи из `docs/01_arifmeticheskie_vychisleniya_updated.md`.
Исходный Markdown не изменяется: все рабочие шаблоны лежат отдельно в
`data/templates/problem_sets/arithmetic/templates.json`.

## JSON-схема

Каждая запись содержит:

- `id` — стабильный идентификатор шаблона;
- `source_problem_numbers` — номера исходных задач, которые покрывает шаблон;
- `source_examples` — примеры исходной формулировки;
- `human_template` — объяснение математической структуры;
- `render_template` — строка с плейсхолдерами для финального условия;
- `parameters` — независимые числовые параметры и диапазоны;
- `derived_parameters` — зависимые параметры, например `base + 1`;
- `constraints` — смысловые ограничения, которые должна сохранять генерация;
- `generation_strategy` — имя Python-стратегии в `problemgen/generation/arithmetic_templates.py`;
- `answer_strategy` — тип вычисления ответа;
- `difficulty` и `tags` — навигационные поля.

В JSON не хранится исполняемый Python-код. Математическая логика живет в
именованных стратегиях генератора.

## Генератор

Основной код находится в `problemgen/generation/arithmetic_templates.py`.

Он умеет:

- загружать и валидировать каталог;
- проверять, что все 75 исходных номеров покрыты ровно один раз;
- генерировать новые числа по seed;
- сохранять связи вроде последовательных чисел, прогрессий, чередующихся знаков
  и распределенного общего множителя;
- считать ответы программно;
- собирать лист из ровно пяти выбранных модулей, случайно выбирая шаблон внутри
  каждого модуля.

## Запуск

Проверить все шаблоны на 100 seed-прогонах:

```bash
python scripts/validate_arithmetic_templates.py
```

Сгенерировать пример JSON-листа из пяти задач:

```bash
python scripts/generate_arithmetic_worksheet.py --seed 12345
```

Запустить сайт:

```bash
python scripts/run_site.py --host 127.0.0.1 --port 8090
```

## Сайт

Сайт использует `problemgen/web/worksheet_site.py` и показывает модули задач, а
не отдельные шаблоны. Сейчас доступен один модуль: `arithmetic`. Пользователь
может выбрать этот модуль в нескольких слотах, например во всех пяти. Для
каждого слота генератор случайно выбирает один из шаблонов внутри модуля,
подставляет новые числа и показывает готовый лист. Ответы скрыты от ученического
листа и доступны отдельным ключом.

## Как добавить шаблон

1. Добавить запись в `data/templates/problem_sets/arithmetic/templates.json`.
2. Если нужна новая математическая логика, добавить именованную стратегию в
   `problemgen/generation/arithmetic_templates.py`.
3. Убедиться, что `source_problem_numbers` не пересекается с другими шаблонами.
4. Запустить `python scripts/validate_arithmetic_templates.py`.
5. Запустить `python -m unittest tests.test_arithmetic_templates`.
