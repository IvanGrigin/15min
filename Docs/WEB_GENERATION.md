# Web Generation

Локальный сайт собирает и печатает учебные листы с математическими задачами.

## Запуск

Из корня проекта:

```bash
python3 scripts/run_site.py --host 127.0.0.1 --port 8090
```

Открыть в браузере: `http://127.0.0.1:8090`.

## Сценарии работы

- **«Сделать готовый вариант»** — быстрый сценарий: по умолчанию создаёт
  пять задач, автоматически выбирает модули и новые числа. В выбор попадают
  только шаблоны с вычисляемыми проверяемыми ответами.
- **Количество задач** — регулируется перед генерацией; локальный сайт
  принимает от 1 до 20 задач.
- **Настроить модули вручную** — открывает отдельный список модулей для
  каждого номера. Один модуль можно повторять.
- **Печатать с ответами** — печатает A4 в альбомной ориентации по образцу
  учебного листа: компактная шапка, задания с разделителями и отдельная правая
  колонка ответов с пунктиром для отрезания ножницами. Пять задач с одной
  рабочей строкой после каждой помещаются на один лист.

## Каталоги задач

`GET /api/templates` объединяет два явно помеченных набора:

- 215 шаблонов из `data/templates/problem_sets/` с работающими генераторами и
  вычисляемыми ответами. Они используются в быстром варианте и подходят для
  печати ответов.
- 1088 очищенных текстовых шаблонов из
  `data/templates/all_tasks_templates.json`. Они доступны как ручной
  архивный модуль, но не участвуют в быстром варианте: у них ещё нет
  восстановленных `answer_formula`, поэтому сайт не выдумывает ответы.

Исключённые при текстовой очистке фрагменты остаются в
`data/templates/all_tasks_templates_rejected.json` и не показываются в UI.
Исходный корпус `Docs/all_tasks_all_files.md` не редактируется.

## HTTP-контракт

### `GET /api/templates`

Возвращает модули, их размеры, разделение проверяемого и архивного каталогов,
а также границы количества задач.

### `POST /generate`

Быстрый вариант:

```json
{"mode": "random", "count": 5, "seed": 20260715}
```

Ручной вариант:

```json
{
  "count": 5,
  "module_ids": [
    "arithmetic",
    "equations",
    "systems_of_equations",
    "comparison_of_numbers_and_expressions",
    "arithmetic"
  ],
  "seed": 20260715
}
```

`count` должен совпадать с длиной `module_ids`. Ответ, текст и сгенерированные
значения передаются раздельно в `selected_templates`.

## Связанные файлы

- `frontend/worksheet_site.js` — динамический выбор числа задач, быстрый и
  ручной запросы, предпросмотр и печать;
- `frontend/worksheet_site.css` — Lato, сине-белая фирменная тема и правила
  A4 landscape с отрезаемой колонкой ответов;
- `assets/logo_239.png` — предоставленный знак «Поступление в 239», который
  используется в шапке сайта и печатного листа;
- `assets/qr.png` — QR-код Telegram-канала `https://t.me/postuplenie239` с
  белой тихой зоной для корректного сканирования;
- `problemgen/web/worksheet_site.py` — HTTP-слой и объединение каталогов;
- `assets/fonts/` — локальные файлы Lato, используемые без CDN.
