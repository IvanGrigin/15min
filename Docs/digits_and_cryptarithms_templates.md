# Шаблоны цифр, записи чисел и криптарифмов

Модуль `digits_number_notation_and_cryptarithms` объединяет два read-only
источника `docs/08a_...md` и `docs/08b_...md`. Каталог находится в
`data/templates/problem_sets/digits_number_notation_and_cryptarithms/templates.json`,
а исполняемая логика — в `problemgen/generation/digits_templates.py`.

Каталог различает вхождения цифры и количество чисел, содержащих цифру.
Точные стратегии включают DP по сумме цифр, конечный поиск пропущенных цифр,
перестановки мультимножеств, подсчёт подпоследовательностей, удаление цифр и
конструируемые сложения со звёздочками. Неактивные записи сохраняют полное
покрытие источников и никогда не выбираются сайтом.

Именованные задачи загружают персонажей только из
`docs/approved_dimensions_150_characters.md`: все роли одной задачи принадлежат
одной вселенной и различаются. Формулировки используют безопасный именительный
падеж; прошедшее время согласуется с родом.

Проверки:

```powershell
py -3 -m unittest tests.test_digits_templates
py -3 scripts/validate_digits_templates.py
```
