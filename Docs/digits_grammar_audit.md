# Грамматический аудит модулей с персонажами

Дата проверки: 2026-07-17.

Проверены активные character-шаблоны модулей arithmetic, equations, systems,
comparisons, sequences, interval counting, divisibility и digits. В первых
трёх и в interval counting именованных шаблонов нет. Для остальных выполнено
по 100 deterministic-прогонов каждого шаблона:

- comparisons: 1 шаблон, 100 экземпляров;
- sequences: 2 шаблона, 200 экземпляров;
- divisibility: 3 шаблона, 300 экземпляров;
- digits: 5 шаблонов, 500 экземпляров.

Проверялись заполненность текста, уникальность ролей, единая вселенная,
согласование мужских и женских форм прошедшего времени и безопасные конструкции
для составных/нарицательных имён. Новых дефектов в прежних модулях не найдено;
их математическая логика не менялась. Digits-модуль использует именительный
падеж в потенциально небезопасных местах и не пытается склонять имя эвристикой.
