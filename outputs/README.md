# Выходные данные

Папка `outputs/` хранит результаты работы генераторов.

Правило:

- сгенерированные файлы не смешивать с исходным кодом;
- повторяемые демонстрационные примеры можно хранить в `outputs/samples/`;
- массовые временные выгрузки складывать в отдельные подпапки по типу задач.

Текущие подпапки:

- `samples/` — сохраненные примеры JSON;
- `friendship_class/` — JSON-наборы задач по сюжету про класс.
